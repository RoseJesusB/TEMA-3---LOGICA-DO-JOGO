# Tema 3 – Desenvolvendo a Lógica do Jogo

Trabalho individual da disciplina de Análise e Desenvolvimento de Sistemas.

## Objetivo
Desenvolver a lógica básica de um jogo utilizando programação,
aplicando conceitos como entrada de dados, decisões e estruturas condicionais.

## Descrição
O projeto apresenta um programa simples em Python que interage com o usuário,
recebe informações pelo teclado e executa decisões com base nas escolhas feitas.

## Tecnologias utilizadas
- Python

## Como executar
1. Abra o arquivo principal do projeto
2. Execute o código em um ambiente Python
3. Siga as instruções exibidas no terminal
